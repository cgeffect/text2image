# 第一课：跑通第一张图

## 目标

- 知道 SD 的最小工作链路
- 能用 Python 脚本生成第一张图
- 理解 prompt / negative prompt 的基本作用

## 先决条件

- 已有 `sd-learn` 环境
- 已有脚本：`python-sd/generate_image.py`
- 网络可下载模型（首次运行）

## 步骤

1. 激活环境：

```bash
source /opt/miniconda3/etc/profile.d/conda.sh
conda activate /Users/tfwang/LLM/sd/sd-learn
```

2. 打开 `python-sd/generate_image.py`，只改这几项：
- `prompt`
- `negative_prompt`
- `seed`

3. 直接运行脚本，等待输出图片到 `outputs/`。

## 本课你需要理解的概念

- **prompt**：你想要什么
- **negative_prompt**：你不想要什么
- **seed**：固定随机性，便于复现
- **model**：决定画风和能力上限

## 作业

- 生成人像 2 张（不同 seed）
- 生成风景 2 张（不同 prompt）
- 保存你最满意的一张并写 2 句复盘

## 验收标准

- 能稳定生成图片，不报错
- 知道改哪几个参数能改变结果
