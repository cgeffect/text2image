# 第四课：LoRA 与风格一致性

## 目标

- 理解 LoRA 是什么、解决什么问题
- 学会加载 LoRA 并控制权重
- 做出风格一致、角色一致的结果

## LoRA 是什么

LoRA 是“轻量增量权重”，常用于：
- 固定角色脸型/服饰风格
- 固定画风（写实、动漫、胶片感）
- 在不改基础模型的情况下快速增强效果

## 关键参数

- `lora_path`：LoRA 文件路径
- `lora_scale`：LoRA 强度（常见 0.4~1.0）
- prompt 触发词：有些 LoRA 需要指定关键词

## 推荐实验法

固定 seed，做 `lora_scale` 梯度实验：
- 0.4
- 0.7
- 1.0

观察：
- 风格是否更统一
- 细节是否过拟合/失真

## 常见坑

- LoRA 与基础模型不兼容
- 权重过高导致“塑料感”或画面崩坏
- 忘记触发词导致效果很弱

## 作业

- 选 1 个风格 LoRA，完成 4 张同风格图
- 选 1 个角色 LoRA，完成 4 张不同场景但角色一致图

## 验收标准

- 你能说清楚：为什么选择这个 LoRA 和这个权重
