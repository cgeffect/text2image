# 第七课：生产化流程与面试表达

## 目标

- 从“会玩”升级到“可交付”
- 建立可复现、可维护的生产流程
- 形成面试可讲的项目故事

## 生产化最小闭环

1. **输入规范化**：prompt 模板、命名规范  
2. **参数可复现**：记录模型、seed、steps、cfg、分辨率  
3. **输出可追踪**：图片文件名和元数据绑定  
4. **质量评审**：每批次人工挑选 + 复盘  
5. **版本管理**：按日期和主题归档

## 你该建立的目录

```text
project/
  prompts/
  outputs/
  logs/
  workflows/
  notes/
```

## 你要会回答的面试问题

- 为什么选 SD1.5 或 SDXL？
- 你如何做参数调优？
- LoRA / ControlNet 分别解决了什么问题？
- 如何保证结果可复现？
- Mac 上如何做性能优化？

## 面试叙述模板（简版）

> 我在本地 Mac 上完成了 Stable Diffusion 部署与实践，使用 Python 和 ComfyUI 双路径生成图片与短视频。  
> 我建立了可复现流程（固定 seed、参数记录、工作流归档），并通过 LoRA 做风格一致性、ControlNet 做构图控制。  
> 最终可以稳定输出成图和短视频，并能针对速度与质量做权衡优化。

## 毕业作业（必须做）

- 一组图片作品（至少 10 张，2 个主题）
- 一个短视频作品（2~5 秒）
- 一份复盘文档（1~2 页）

## 验收标准

- 你能独立从 0 到 1 完成一次生成任务
- 你能清楚解释每个关键技术点
