# 人物肖像 Prompt 模板（学习版）

以下模板优先用于 SD1.5，本地低资源环境也比较稳定。

## 1) 写实商务肖像

**Prompt**
`专业人像摄影，头肩部构图，人物居中，五官对称，皮肤纹理自然，柔和棚拍光，清晰对焦，85mm镜头感`

**Negative**
`畸形，人体结构错误，眼睛不对称，斗鸡眼，脸部变形，多余手指，手部畸形，模糊，低清晰度，水印，文字，logo`

## 2) 户外阳光肖像

**Prompt**
`户外人像摄影，头肩部构图，坐在阳光草坪上，金色逆光，皮肤纹理自然，清晰对焦，浅景深`

**Negative**
`畸形，人体结构错误，眼睛不对称，脸部变形，多余手指，手部畸形，模糊，低清晰度，森林背景，水印，文字，logo`

## 3) 电影感夜景肖像

**Prompt**
`电影感人像摄影，头肩部构图，柔和轮廓光，轻微霓虹背景，皮肤纹理真实，清晰对焦，高对比度光线`

**Negative**
`畸形，人体结构错误，眼睛不对称，脸部变形，多余手指，手部畸形，模糊，低清晰度，面部噪点，水印，文字，logo`

## 使用建议

- 先固定 seed，再改 prompt 做对比
- 先用 `safe` 或 `standard` 档位
- 人像容易变形时，优先降低 cfg（如 6.8 -> 6.2）并提高 steps（如 20 -> 24）
